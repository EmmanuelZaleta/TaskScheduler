INSTRUCCIONES PARA EL ICONO
============================

El instalador requiere un archivo icon.ico para mostrar un icono profesional.

OPCIONES PARA CREAR EL ICONO:

1. Usar el script de Python (Recomendado para pruebas rápidas):
   - Instalar Pillow: pip install Pillow
   - Ejecutar: python create-icon.py
   - Esto creará un icono básico con la letra "Y"

2. Crear un icono profesional online:
   - Visitar: https://www.favicon-generator.org/
   - O: https://www.icoconverter.com/
   - Subir un logo o imagen de 512x512 px o mayor
   - Descargar como .ico con múltiples resoluciones
   - Guardar como "icon.ico" en este directorio

3. Usar herramientas de diseño:
   - GIMP (gratis): https://www.gimp.org/
   - Adobe Photoshop
   - Inkscape (gratis, para vectores): https://inkscape.org/

ESPECIFICACIONES DEL ICONO:

- Formato: .ico (Windows Icon)
- Resoluciones recomendadas dentro del .ico:
  * 16x16 px
  * 32x32 px
  * 48x48 px
  * 64x64 px
  * 128x128 px
  * 256x256 px

- Fondo: Transparente o usar el color corporativo #2E75B6

MIENTRAS TANTO:

Si no tiene un icono listo, el instalador usará el icono predeterminado de Windows.
El instalador se puede construir sin el icono, pero se mostrará una advertencia.

NOTA: Para un producto profesional, se recomienda encargar el diseño del icono
a un diseñador gráfico profesional.
